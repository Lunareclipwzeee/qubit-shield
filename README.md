# EIGENLOCK

> WARNING: Pre-audit software. Not formally audited. Suitable for pilots and evaluation only.

India first post-quantum cryptography SDK. Built on ML-KEM-768 (NIST FIPS 203).

## What This IS
- Real ML-KEM-768 NIST FIPS 203 key encapsulation
- Real AES-256-GCM authenticated encryption  
- Real syndrome-based tamper detection
- Production API with signup dashboard usage tracking

## What This Is NOT
- A quantum computer
- Physical qubit simulation
- Hardware quantum cryptography
